Self-Closing Tags

Self-closing tags, also known as void elements or empty elements, are
HTML tags that do not require a closing tag. They represent elements
that do not contain any content or children. Instead of using a
separate closing tag, self-closing tags are written with a forward
slash at the end of the opening
Examples of self-closing tags in HTML include:

: Line break tag : Image tag : Input field tag : Meta information tag

Meta Tags

Meta tags are special HTML tags used to provide metadata or
additional information about an HTML document. They are placed
within the section of an HTML document and are not visible
to website visitors. Meta tags provide information to browsers,
search engines, and other web services about the webpage's content,
encoding, authorship, keywords, and more.
