Global Attributes

Global attributes in HTML are attributes that can be used on any
HTML element, regardless of its type. These attributes provide
common functionality and behavior across different elements.
Here are five commonly used global attributes:

1. class:Specifies one or more class names for an
element, allowing you to target and style elements using CSS or
JavaScript.
2. id: Provides a unique identifier for an element.
It can be used for targeting specific elements with CSS or JavaScript.
3. style: Allows inline CSS styling for an element.
You can define CSS properties and values directly within the attribute.
4. title: Provides additional information or a tooltip
text that is displayed when the user hovers over an element.
5. dataAllows custom data attributes to be added to
HTML elements. These attributes can store extra information that can
be accessed using JavaScript.
